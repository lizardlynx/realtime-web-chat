# realtime-web-chat
## Short Description
University project, which is focused on creating a realtime-web-chat.  
## Concept
Chat provides possibility to talk to people, who use the same app.
## Functionality
* talking to everyone in big group chat;
* search function;
* separate rooms;
* choose name/nickname;
* choose photo.
